# TidyTuesday: The Rise of Running Quarterbacks in Modern Football

## Overview
Scatter plot visualization showing NFL quarterbacks rushing performance from 2000 to 2017. Each point represents a quarterback's career stats, with x-axis as total rushing touchdowns and y-axis as average yards per attempt. Highlights standout running QBs like Michael Vick and Cam Newton in blue.

## Key Findings
- Running QBs cluster low on TDs but high on yards per attempt efficiency.
- Stars like Vick, Newton, and Kaepernick dominate the top-right quadrant.
- Traditional pocket passers form a dense cloud near origin, showing era shift.
- Average performance sits around 2.5 yards per attempt with few TDs.
- Modern trend visible: more QBs above average line post-2010.

## Data Source
NFL stats dataset via [TidyTuesday](https://github.com/rfordatascience/tidytuesday/tree/main/data/2018/2018-08-28)

## Tools
R (tidyverse, ggplot2, TidyTuesdayR, ggtext, scales, ggrepel, showtext)
