# TidyTuesday: NFL Pay Disparity (2011-2018)

## Overview
This visualization explores salary inequality across different NFL positions from 2011 to 2018, measured by the gap between mean and median salaries.

## Key Findings
- Quarterback has the highest salary inequality gap among all positions
- In 2018, Offensive Lineman had the lowest salary inequality gap compared to other positions
- There is no specific pattern in pay gap figures across all positions year-over-year

## Data Source
[NFL Salary](https://github.com/rfordatascience/tidytuesday/blob/main/data/2018/2018-04-09/nfl_salary.xlsx) 

## Tools
- R (tidyverse, tidytuesdayR, ggrepel, ggtext, showtext)
- ggplot2 for visualization
